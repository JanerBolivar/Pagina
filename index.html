Carro autómata con IOT.

La cybertruck es un vehiculo automata el cual utiliza un sensor de ultrasonido y un sensor de color para determinar las acciones que va arrealizar, como detectar obstaculos y detectar colores para tomar la decision de girar hacia el lado que le indique.

Conoce mas de el

Decisiones que debe tomar el carro.

El vehículo utiliza un sensor de ultrasonidos, un sensor de colores y un módulo WiFi ESP8266-01s.

Ultrasonido

El sensor emite pulsaciones acústicas cortas a través del aire, y de esta forma detecta objetos que después reflejará. Lo que mide le sensor es el tiempo entre emitir el pulso y la recepción del este.

Colores

El sensor de color converte la luz a una frecuencia que combina fotodiodos de silicio reconfigurables y una corriente de frecuencia en un solo circuito integrado.

WiFi

El ESP8266 es un sistema en chip (SoC) altamente integrado que combina un procesador de microcontrolador de bajo consumo con capacidades de conectividad Wi-Fi.
